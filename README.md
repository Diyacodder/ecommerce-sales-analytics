# E-Commerce Sales Analytics

> An end-to-end Data Analytics project using **Python**, **SQLite**, and **Google Looker Studio**.

---

##  Project Overview

This project analyzes an e-commerce sales dataset to uncover meaningful business insights through data cleaning, exploratory data analysis (EDA), SQL queries, and interactive dashboarding.

The project demonstrates the complete analytics workflow, from raw data to business recommendations.

---

##  Objectives

- Clean and preprocess raw sales data
- Perform Exploratory Data Analysis (EDA)
- Analyze business performance using SQL
- Build an interactive dashboard
- Generate actionable business insights

---

##  Tech Stack

##  Key Insights

- Office Supplies generated the highest sales.
- Office Supplies generated the highest profit.
- California recorded the highest sales.
- The West region was the top-performing region.
- Phones and Chairs were among the top-selling products.
- Furniture received the highest average discount.
